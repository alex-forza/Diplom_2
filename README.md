# Diplom_2
1. data содержит в себе используемые классы: StaticData, ExpectedResults, Api для удобной работы.
2. helpers содержит функции для генерации почты, пароля, имени, payload.
3. requirements.txt содержит внешние зависимости.
4. создан .gitignore.
5. создан allure-отчёт.
6. сами тесты описаны в директории test:
6.1. test_create_user создает уникального, уже зарегистрированного юзера, а также создает пользователя без заполенния обязательного поля.
6.2. test_login_user логинит под существующим юзером, а также логинит с неверным логином и паролем.
6.3. test_change_data изменяет данные юзера с авторизацией и без.
6.4. test_create_order создает заказ с авторизацией и без, с ингредиентами и без, с неверным хэшем.
6.5. test_get_orders проверяет заказы авторизованного и неавторизованного пользователей.
7. conftest.py содержит функцию-фикстуру по возврату случайного ингредиента.
